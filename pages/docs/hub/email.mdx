# Email Inbox

This page guides you through setting up the Email inbox.

## How to set up an Email channel?

To enable the Email Channel, you'll need to turn on IMAP and create an app-specific password in your email account. This example demonstrates the steps for Gmail.

### Step 1: Enable IMAP
Before you start, make sure IMAP access is enabled in your [Gmail settings](https://mail.google.com/mail/u/0/#settings/fwdandpop).

<Frame>
  ![Enable IMAP](/images/hub/emailStep01.png)
</Frame>

### Step 2: Create an App-Specific Password
Create an app-specific password and save it.

- Go to [Google Account Security](https://myaccount.google.com/security)
<Frame>
  ![Google Account Security](/images/hub/emailGoogle1.png)
</Frame>
- Enable Two-step verification
<Frame>
  ![Google Account Security](/images/hub/emailGoogle02.png)
</Frame>
- Click "App passwords" under the "Signing in to Google" section on the Security page.
<Frame>
  ![Google Account Security](/images/hub/emailGoogle2.png)
</Frame>
- Select the Mail app, select a device, and click "GENERATE". Use the generated password to configure IMAP/SMTP.
<Frame>
  ![Google Account Security](/images/hub/emailGoogle3.png)
</Frame>

For detailed instructions, refer to this [guide](https://knowledge.workspace.google.com/kb/how-to-create-app-passwords-000009237).

### Step 3: Add Inbox in Engagement Hub
Navigate to **Engagement Hub** → **Settings** → **Inboxes** → **Add Inbox**.

<Frame>
  ![Add Inbox](/images/hub/telegramStep1.png)
</Frame>

### Step 4: Select the Email Icon
Select the **Email** icon.

<Frame>
  ![Select Email Icon](/images/hub/emailStep1.png)
</Frame>

### Step 5: Choose the Other Icon
Choose the **Other** icon.

<Frame>
  ![Choose Other Icon](/images/hub/emailStep2.png)
</Frame>

### Step 6: Fill in the Required Fields
Fill in the required fields in the form and click **Check Connection**. If successful, the **Save** button will become enabled.

#### SMTP
- **From Email:** The email address you wish to connect.
- **Password:** The app-specific password created earlier.
- **Host:** The SMTP host of the email service you are using.
- **Port:** The SMTP port of the service.

#### IMAP
- **Host:** The IMAP host of the service you are using.
- **Port:** The IMAP port of the service.

</Frame>
