# SMS twilio Inbox

This page guides you through setting up the SMS inbox

## How to setup a SMS channel?

To use SMS Channel, you have to create a Twilio app in the Twilio Studio. You can find more details about creating Twilio account [here](https://www.twilio.com/docs).

###### Step 1. Go to Engagement Hub → Settings → Inboxes → “Add Inbox”.
<br/>
<Frame>![telegram Step 1](/images/hub/telegramStep1.png)</Frame>

###### Step 2. Click on the "SMS" icon.
<br/>
<Frame>![telegram Step 1](/images/hub/twilio1.png)</Frame>

###### Step 3. Choose your API provider and configure the inbox. Explained below are both options.

If you want to use Twilio to set up your SMS inbox, choose that from the dropdown. It will display the required fields as explained below.
<br/>
<Frame>![telegram Step 1](/images/hub/twilio2.png)</Frame>

These are the inputs required to create this channel:
<br/>
<Frame>![Step](/images/hub/twilio.png)</Frame>

Click Create Twilio Channel after filling in the information.
