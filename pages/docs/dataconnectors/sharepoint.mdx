# Sharepoint

This page guides you through setting up the Sharepoint data connector to sync data and keep your files up-to-date with agents.

## Prerequisite
To set up the Sharepoint data connector, you'll need you to create and register application in Azure Entra.

### Set up the Sharepoint connector
Log into your Google account.
On the Set up the source page, select Amplitude from the Source type dropdown.
Click Set up source.
