# Zendesk

This page guides you through setting up the Zendesk data connector to sync data.

## Prerequisite

1.  Login to [Zendesk](https://www.zendesk.com/login/)
2.  Click on **Settings** menu icon and select **Apps and integrations**
3.  Choose **Zendesk API**
4.  Create an API Token

### Set up the Confluence connector

1. Register an application  
    i. Name  
   ii. Zendesk subdomain: This is your zendesk home page which is after **https://** and before with **.zendesk.com/**  
    i. E-mail: This is a Zendesk login email
   ii. API token: This is your API Token which you have created

2. Connect
3. Press the button **Create**
