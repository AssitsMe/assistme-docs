# Confluence

This page guides you through setting up the Confluence data connector to sync data.

## Prerequisite
 1. Login to [Atlas](https://id.atlassian.com/login?continue=https%3A%2F%2Fhome.atlassian.com%2F&application=atlas/)  
 2. Click upper-left menu icon and select **Confluence**  
 3. Create a **space** if needed  
 4. Add **pages** to space  
 5. Create an API token [here](https://id.atlassian.com/manage-profile/security/api-tokens)  

### Set up the Zendesk connector
  1. Register an application  
       i. Name  
      ii. Confluence url: This is your confluence home page which ends with **/wiki**  
        i. E-mail: This is a Confluence login email
       ii. API token: This is your API Token which you have created 

 2. Connect  
 3. Select **Space**  
 4. Select **Pages**. You can choose one or more pages  
 5. Press the button **Create**
